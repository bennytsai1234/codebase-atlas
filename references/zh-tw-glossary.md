# Traditional Chinese Glossary

Use this reference only when the selected atlas output language is Traditional
Chinese. Keep code identifiers, file paths, commands, API names, package names,
and product names unchanged.

## Standard Terms

- Codebase Atlas: Codebase Atlas
- Module Index: 模組索引
- Purpose And Usage: 目的與使用方式
- Module List: 模組列表
- Module Summaries: 模組摘要
- Workflow Docs: 工作流程文檔
- Workflow Entry Adapters: 工作流程入口適配器
- Workflow Entrypoints: 工作流程入口
- Current State: 目前狀態
- Scope: 範圍
- Upstream Dependencies: 上游依賴
- Downstream Impact: 下游影響
- Key Flows: 主要流程
- Common Change Entry Points: 常見變更入口
- Known Risks: 已知風險
- Do Not Do: 不要做
- Target Current State: 目標專案目前狀態
- Target Upstream/Downstream: 目標專案上下游
- Reference Counterpart: 參考對應
- Useful Patterns: 可參考模式
- Target Change Entry Points: 目標專案變更入口
- Introduction Workflow: 介紹工作流程
- Bug Workflow: Bug 工作流程
- Feature Workflow: 功能工作流程
- Optimization Workflow: 優化工作流程
- Investigation Workflow: 調查工作流程
- Refactor Workflow: 重構工作流程
- Validation Workflow: 驗證工作流程
- Before / After Gate: 修改前／修改後閘門
- Before: 修改前
- After: 修改後
- Findings Summary: 調查摘要
- Validation Summary: 驗證摘要
- Delivery Policy: 交付政策

## Tone

- Prefer concise, engineering-focused Traditional Chinese.
- Use `模組` for module, `流程` for flow, `邊界` for boundary, `依賴` for
  dependency, `驗證` for validation, and `風險` for risk.
- Avoid translating stable project names, package names, command names, and file
  paths.
